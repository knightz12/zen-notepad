# Zen Notepad

Zen Notepad is a modern lightweight text editor inspired by the simplicity of Windows Notepad, redesigned with a cleaner interface, tab support, and productivity-focused features.

Built with Electron, Zen Notepad keeps the fast and minimal experience of classic Notepad while adding modern workflow improvements for writing, coding, note-taking, and editing plain text files.

---

## Features

* Modern dark-themed UI
* Multi-tab editing
* Fast lightweight performance
* Find & Replace
* Word Wrap toggle
* Auto-save session recovery
* Unsaved changes protection
* Keyboard shortcuts
* Sidebar pin mode
* Single-window tab system
* Support for multiple text/code file formats
* Windows-style UTF-8 + CRLF saving
* TXT file association support

---

## Supported File Types

Zen Notepad supports opening and editing:

```text
txt, log, md, json, js, ts, html, css,
xml, csv, py, java, c, cpp, h, hpp,
ini, bat, cmd, ps1, yaml, yml
```

---

## Keyboard Shortcuts

| Shortcut           | Action       |
| ------------------ | ------------ |
| Ctrl + N           | New File     |
| Ctrl + O           | Open File    |
| Ctrl + S           | Save         |
| Ctrl + F           | Find         |
| Ctrl + H           | Replace      |
| Ctrl + W           | Close Tab    |
| Ctrl + Tab         | Next Tab     |
| Ctrl + Shift + Tab | Previous Tab |
| Ctrl + P           | Pin Sidebar  |

---

## Why Zen Notepad?

Zen Notepad is designed for users who want the simplicity of Windows Notepad but with modern usability improvements:

* cleaner UI
* tabs
* better file workflow
* session recovery
* improved editing experience

without becoming a heavy IDE.
