# Zen Notepad

Zen Notepad is a modern lightweight text editor inspired by Windows Notepad, redesigned with a cleaner dark UI and productivity-focused features.

Built with Electron, Zen Notepad keeps the simplicity of classic Notepad while adding a smoother and more modern experience for everyday writing, coding, quick notes, and text editing.

## Features

* Modern dark-themed interface
* Multi-tab editing
* Fast lightweight performance
* Find and Replace
* Word Wrap toggle
* Auto-save session recovery
* Custom unsaved changes prompt
* Keyboard shortcuts
* Sidebar pin mode
* TXT file support and file association
* Clean distraction-free writing area

## Keyboard Shortcuts

* Ctrl + N → New File
* Ctrl + O → Open File
* Ctrl + S → Save
* Ctrl + F → Find
* Ctrl + H → Replace
* Ctrl + W → Close Tab
* Ctrl + P → Pin/Unpin Sidebar
* Ctrl + Tab → Next Tab
* Ctrl + Shift + Tab → Previous Tab

## Why Zen Notepad?

Zen Notepad keeps the familiar workflow of Windows Notepad but improves the experience with:

* modern visuals
* tab support
* cleaner organization
* better workflow controls
* persistent sessions

It is designed for users who want a minimal editor without the complexity of large IDEs.
